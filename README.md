# Install
1. Install the  theme from the [Marketplace](http://127.0.0.1).
2. Open the command palette (`Ctrl/Cmd + Shift + P`) and search for `Preferences: Color Theme` or use  (`Ctrl/Cmd + K + D`) keyboard shortcut.
3. Select `DevoraBB`.
